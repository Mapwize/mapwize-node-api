Create a MapwizeApi client
Parameters:
Name Type Description
apiKey String the Mapwize API key to use. API keys can be found in the Mapwize admin interface under the Application menu
organizationId String the id of your organization. For now, the use of the API is limited to your organization.
opts Object an object with optional parameters serverUrl the server url to use. Default to production server at https://api.mapwize.io
